# Experiment 1.1: Front-end Application with React

This project demonstrates advanced front-end development techniques using React, focusing on state management, UI libraries, and performance optimization.

## Key Features Implemented:
* **Part A: Context API & Memoization** - Global state management with re-rendering optimization.
* **Part B: Redux Toolkit** - Advanced state management using slices and selectors.
* **Part C: UI Integration & PWA** - Integration of Material UI, Ant Design, and PWA manifest.
* **Part D: Custom Hooks** - `useThemeSwitcher` hook for optimized theme toggling.

## Technology Stack:
* React (Vite)
* Redux Toolkit
* Material UI & Ant Design
* Styled Components

## How to Run:
1. `npm install --legacy-peer-deps`
2. `npm run dev`
